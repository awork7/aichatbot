**SAI (Secure AI Banking Assistant)** is an AI-powered conversational agent built for localized financial and banking sector queries using a RAG (Retrieval-Augmented Generation) approach. It operates completely offline, ensuring maximum data security, compliance, and privacy for sensitive institutional data.

## 🌟 Features

- 🔒 **100% Offline Operation** - Zero internet connectivity required after initial model setup.
- 🏦 **Domain-Specific Knowledge Base** - Ingests localized banking rules, loan eligibility structures, and product PDFs.
- 🧠 **RAG Architecture** - Intelligently retrieves relevant financial context chunks before generating precise responses.
- 💰 **Zero Operational Cost** - Completely free to run locally using open-source Large Language Models (LLMs).
- 🖥️ **Clean Web UI** - Professional Streamlit-based interactive dashboard.
- 🔐 **Privacy-First Design** - No API keys, no data leakage, and all training/retrieval vectors stay strictly on-premise.
- ⚡ **Hardware Optimized** - Tuned to run efficiently on standard corporate environments with 16GB RAM.
